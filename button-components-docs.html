Button Components

Overview

Button Types

Eight button types organized by visual hierarchy and intent. Each type supports three sizes, icon placement, and full-width modes. White and White Tertiary are specifically designed for use on dark or brand-colored surfaces.

Transactional

Revenue & conversion actions

Use for actions that directly drive revenue or conversion: Buy Tickets, Purchase, Add to Cart, Subscribe. This is the highest-emphasis button. Should be used sparingly — typically only one per view for the primary conversion action.

<button class="btn btn-transactional btn-700">Buy Tickets</button> <button class="btn btn-transactional btn-300 btn-icon-leading"> <svg class="btn-icon">...</svg> <span>Buy Tickets</span> </button>

Primary

Main page actions

Use for the primary action on a page or within a context: Submit, Save, Confirm, Continue. Uses the team's brand-interactive color. Limit to one primary button per view section to maintain hierarchy.

<button class="btn btn-primary btn-300">Save</button>

Neutral

High-contrast, non-branded

Use for high-contrast actions that should not carry brand emphasis. Uses neutral-1000 (black in light mode, white in dark mode). Good for sign-up flows, modals, or areas where brand color competes with content.

<button class="btn btn-neutral btn-300">Learn More</button>

Secondary

Outlined supporting actions

Use alongside primary buttons for supporting actions: Cancel, Back, View Details. The outlined style communicates lower emphasis than filled buttons while remaining clearly interactive.

<button class="btn btn-secondary btn-300">Cancel</button>

Tertiary

Low-emphasis text actions

Use for low-emphasis actions within content: links in context, optional actions, "Skip" or "Not now" options. No fill or border in default state — appears as styled text that gains a background on hover.

<button class="btn btn-tertiary btn-300 btn-icon-trailing"> <span>View All</span> <svg class="btn-icon">...</svg> </button>

Destructive

Irreversible & dangerous actions

Use for actions that are permanent or dangerous: Delete, Remove, Revoke Access. The red fill signals caution. Always pair with a confirmation dialog for irreversible actions.

<button class="btn btn-destructive btn-300">Delete</button>

White

CTA on dark & brand surfaces

Use for high-contrast CTAs placed on dark backgrounds, brand-colored hero sections, or image overlays. Fill: white1000, Text: black1000. Uses the fillWhite interactive surface for hover/pressed states.

<!-- Use on dark/brand backgrounds --> <div style="background: var(--brand-core);"> <button class="btn btn-white btn-300">Get Started</button> </div>

White Tertiary

Low-emphasis on dark surfaces

Use for low-emphasis actions on dark or brand-colored surfaces: secondary links, "Learn More", "Skip" on dark backgrounds. Transparent fill with white text, gains a subtle background on hover. Uses the ghost interactive surface.

<!-- Use on dark/brand backgrounds --> <div style="background: var(--brand-core);"> <button class="btn btn-white-tertiary btn-300">Learn More</button> </div>
