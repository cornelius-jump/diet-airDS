Welcome to Diet AirDS

This is a pure CSS version of AirDS that is easy to understand and great for prototyping. Copy and paste these prompts to build with AI code generators. Learn More about Diet AirDS

How to Use

Copy any prompt below and paste it into an AI code generator like Claude, ChatGPT, or similar tools. Replace [Describe what you want to build] with your component description.

Tip: Change data-mode="light" to data-mode="dark" for dark mode variants.
