Forms

Text inputs, select dropdowns, all interaction states, and tag & chip components. The input system handles focus, error, disabled, and has-value states automatically via modifier classes.

Inputs

Single-line text inputs and select dropdowns with full state, variant, and modifier support. All inputs share consistent structure, styling, and focus states.

Focus color: Input fields use --org-primary-button for focus border, label, and icon colors. Select placeholder text uses --text-secondary.

Input Field Structure

Every input follows a consistent wrapper structure:

<div class="input-field"> <!-- Label row (optional) --> <div class="input-label-row"> <label class="input-label">Label</label> <a class="input-link">Optional link</a> </div> <!-- Input and message wrapper --> <div class="input-and-message"> <div class="input-control"> <span class="input-icon material-symbols-rounded">icon</span> <input type="text" placeholder="Placeholder"> <button class="input-clear material-symbols-rounded">close</button> </div> <p class="input-message">Helper or error message</p> </div> </div>

Select Dropdowns

Add .input-select modifier to .input-field. Uses native <select> as invisible overlay for accessibility.

<div class="input-field input-select" id="select-1"> <div class="input-and-message"> <div class="input-control"> <span class="input-select-display is-placeholder" id="sel-1-display"> Choose option </span> <span class="input-select-chevron material-symbols-rounded"> arrow_drop_down </span> <select id="sel-1" onchange="syncSelect('select-1', 'sel-1-display', this)" onfocus="openSelect('select-1')" onblur="closeSelect('select-1')"> <option value="" disabled selected>Choose option</option> <option value="1">Option 1</option> </select> </div> </div> </div>

Icon choice: Select dropdowns use arrow_drop_down icon (not expand_more). The chevron rotates 180° when the dropdown is open.

Required JavaScript Helpers

Input fields with clear buttons or select dropdowns need these helpers:

// Text input with clear button function syncHasValue(fieldId, input) { const field = document.getElementById(fieldId); field.classList.toggle('has-value', input.value.trim()); } function clearInput(fieldId, inputId) { const field = document.getElementById(fieldId); const input = document.getElementById(inputId); input.value = ''; field.classList.remove('has-value', 'is-error'); input.focus(); } // Select dropdown function syncSelect(fieldId, displayId, selectEl) { const display = document.getElementById(displayId); const chosen = selectEl.options[selectEl.selectedIndex]; if (chosen && chosen.value) { display.textContent = chosen.text; display.classList.remove('is-placeholder'); } else { display.textContent = chosen.text; display.classList.add('is-placeholder'); } } function openSelect(fieldId) { document.getElementById(fieldId).classList.add('is-open'); } function closeSelect(fieldId) { document.getElementById(fieldId).classList.remove('is-open'); }

Tags & Chips

Tags are static label badges for metadata, categories, and status. Chips are interactive filter/toggle controls for filter bars and selection UIs. Both are 32px height with 4px border radius.
